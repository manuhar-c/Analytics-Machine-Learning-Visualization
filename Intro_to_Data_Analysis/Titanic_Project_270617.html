#Q1 : Defining a function which returns an age range depending on the value

def age_range(x):
    if 0 <= x['Age'] <= 15 :
        return 'Child (0-15)'
    elif 16 <= x['Age'] <= 25:
        return 'Young (16-25)'
    elif 26 <= x['Age'] <= 50:
        return 'Adult (26-50)'
    elif x['Age'] >51:
        return 'Aged (50+)'
    
#Applying this function to create a new column in the dataframe
t_df['Age_Range']=t_df.apply(age_range, axis='columns')

Q4 Link between Travelling Alone & Survival Rate

For answering this question, I would need to know whether the passenger was travelling alone or in a group. Hence, I will create a new column, which will depend on the values of SibSp and Parch. If both these columns (SibSp, Parch) are 0, this would imply that the passenger was travelling alone. (this ignores the exceptional cases of children travelling with nannies or people accompanied by mistresses)

In [13]:
#Q4 Defining a function which returns 1 or 0 to travelling alone column
def passenger_alone(x):
    if x['SibSp'] ==0 and x["Parch"] == 0:
        return 1
    else:
        return 0
#Applying this function to create a new column 'Alone'
t_df['alone']=t_df.apply(passenger_alone, axis='columns')

Limitations¶

This analysis of the data suffers from the following limitations

  1. Certain rows of data with missing age values were removed from the analysis
  2. The travelling alone variable does not take into account passengers who were travelling with nannies or passengers who were travelling with mistresses or finaces
